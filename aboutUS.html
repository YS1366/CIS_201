Project Goal

This is our first major project using HTML, CSS, Bootstrap, and JavaScript. The goal was to create a functional and visually appealing menu and to be able to order (add to cart) items.

I chose to build a restaurant menu as it provides a perfect opportunity to practice JavaScript for dynamic content, event handling, and state management (the shopping cart).

Key Features

  • Dynamic Menu: The menu items are loaded from a JavaScript object, allowing for easy updates. The page content changes based on the selected category.
  • Interactive Cart: Users can add, remove, and adjust the quantity of items in a pop-up modal cart that calculates the total price in real-time.
  • Responsive Design: The site uses Bootstrap 5 to ensure it is fully functional and looks professional on all devices, from mobile phones to desktops.
  • Search Functionality: Users can filter the menu items in the currently selected category using the search bar.

Material used

This project was built from scratch using:

  • HTML5 for the content structure.
  • CSS3 for custom styling (colors, fonts, shadows, and transitions).
  • Bootstrap 5 for the responsive grid, navbar, and modal components.
  • JavaScript (ES6+) to power all the interactivity, including DOM manipulation, event listeners, and cart logic.
